GlasStore - Proyecto de Aprendizaje Frontend
Descripción
GlasStore es un proyecto de aprendizaje desarrollado para el curso de desarrollo frontend. Se trata de una tienda ficticia de lentes de sol que muestra una página web estática con una interfaz sencilla y atractiva. El objetivo principal es practicar y aplicar conocimientos de HTML, CSS y JavaScript para crear una página web funcional y bien estructurada.
La página incluye:

Un header con un logo y un menú de navegación.
Una sección principal con un mensaje de bienvenida y una galería de productos (lentes de sol con imágenes, nombres y precios).
Un footer con enlaces a redes sociales (ficticios, salvo Instagram) y un aviso de derechos reservados.

Este proyecto está diseñado para consolidar habilidades en maquetación web, diseño responsivo y estructura semántica de HTML.
Tecnologías Utilizadas

HTML5: Para la estructura semántica de la página.
CSS3: Para el diseño visual, incluyendo estilos personalizados en el archivo style.css.
JavaScript: (Aún no implementado en este proyecto, pero planeado para futuras mejoras).
Git y GitHub: Para el control de versiones y alojamiento del código.

Instalación y Ejecución

Clonar el repositorio:git clone https://github.com/djinnxs/glasstore.git


Abrir el proyecto:
Navega al directorio del proyecto:cd glasstore


Abre el archivo index.html en un navegador web (puedes hacer doble clic en el archivo o usar un servidor local como Live Server en Visual Studio Code).


Dependencias:
No se requieren dependencias externas. El proyecto usa HTML y CSS puros.
Asegúrate de que las imágenes en la carpeta images/ estén disponibles para que se muestren correctamente.
